Using Github as your repo

  • Go to your terminal or git-bash
  • Create a directory or folder called 'test'
  • Change directory to the created directory using 'cd'
  • Create index.html and style.css
  • Type "touch index.html style.css"
  • Type "ls" to see the listed files
  • Type "git init"
  • Type "git add -A" or "git add ." (include the period in the 2nd option)
  • Type "git commit -m 'my first commit' "
  • Type 'git status' to check status
  • Go to github.io
  • Create an account
  • Create a new repository
  • Select 'HTTPS"
  • On your laptop local directory, copy and paste a url like "git remote add origin https://github.com/username/remote-repo.git" from Github
  • On your laptop local directory, type "git push -u origin master"
  • Type username
  • Type password

To update files after first push

  • Go to the terminal or cmd
  • Go to the local directory
  • Type "git status" to see which files have changed
  • Type "git add -A"
  • Type "git commit -m 'your comment' "
  • Type "git push"
